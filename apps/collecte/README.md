# Collecte Dashboard

Projet simple pour afficher les données collectées par le projet `workers`.

## Caractéristiques
- Affichage en temps réel des mentions (polling 30s)
- Statistiques rapides (Total, Positif, Négatif)
- UI Premium avec Vanilla CSS
- Zéro configuration (utilise le endpoint `/demo/mentions` de l'API)

## Développement
```bash
npm run dev
```

## Production
```bash
npm run build
```
